Manera de compilar:

La manera de compilar y ejecutar es desde la terminal con el siguiente comando

java Main

No requiere argumentos, a la hora de ejecutar el archivo Main se daran las instrucciones paso a paso para poder ejecutar el programa adecuadamente.

Todos los archivos se encuentran en la carpeta src ademas se encuentra la carpeta javadoc donde se encuentran los comentarios.
